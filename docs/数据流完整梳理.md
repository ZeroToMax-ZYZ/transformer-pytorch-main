# Transformer 数据流完整梳理

> 以 `nets/build_transformer.py` 的 `make_model` 为例，超参数取自 `train_multi30k_base.py` 的默认配置。

## 超参数速查表

| 符号 | 含义 | 典型值 |
|------|------|--------|
| B | batch_size | 4（动态） |
| S | 源序列长度 | 12（变长） |
| T_full | 完整目标序列长度（含 `<bos>` + `<eos>`） | 14（变长） |
| T | Decoder 输入长度 = T_full - 1 | 13 |
| vocab_size | 词表大小 | 32000 |
| d_model | 隐藏层维度 | 512 |
| d_ff | FFN 膨胀维度 | 2048 |
| h | 注意力头数 | 8 |
| d_k | 每头维度 = d_model / h | 64 |

---

## 阶段 0：BPE 分词 → DataLoader

原始文本经过 BPE 分词后，每个句子变成一个整数 id 列表：

```
原始:  "A dog is running"  →  BPE tokenization  →  [156, 4821, 87, 3012, 2]
                                                       ↓
                                              token id 序列 (List[int])
```

DataLoader 将一个 batch 的变长序列 padding 到相同长度，输出两个张量：

```
src: (B, S)        dtype=torch.int64   源语言 token id（已 padding）
tgt: (B, T_full)   dtype=torch.int64   目标语言 token id（已含 <bos> 和 <eos>，已 padding）

示例 (B=2, S=6, T_full=7):
src = [[156, 4821,  87, 3012,   2,   0],    ← 最后一位是 PAD
       [ 72,  533, 118,    2,   0,   0]]

tgt = [[  1,  89, 203, 417,   2,   0,   0],  ← <bos>=1, <eos>=2, PAD=0
       [  1, 312,  45,  78,  99,   2,   0]]
```

---

## 阶段 1：Seq2SeqBatch 构造（data/batch.py）

`Seq2SeqBatch.from_tensors(src, tgt, pad_idx=0)` 完成三件事：

### 1.1 构造 src_mask

```python
src_mask = (src != pad_idx).unsqueeze(1)   # (B, S) → (B, 1, S)
```

```
src_mask: (B, 1, S)   dtype=torch.bool

示例:
src_mask = [[[True, True, True, True, True, False]],     ← batch 0
            [[True, True, True, True, False, False]]]    ← batch 1
              ↑ 广播到所有 head: (B, 1, S) → (B, H, S) 在 MHA 内部处理
```

**作用**: 在 Encoder 自注意力和 Decoder 交叉注意力中，屏蔽 PAD 位置。

### 1.2 Shift Right（右移切分）

```python
tgt_input = tgt[:, :-1]   # (B, T_full) → (B, T)    丢掉最后一个 token
tgt_y     = tgt[:, 1:]    # (B, T_full) → (B, T)    丢掉第一个 token (<bos>)
```

```
tgt:        [  <bos>,  89,  203,  417,  <eos>,  PAD,  PAD ]
               ↓ 切分 ↓
tgt_input:  [  <bos>,  89,  203,  417,  <eos>,  PAD ]     ← 送入 Decoder
tgt_y:      [    89,  203,  417,  <eos>,  PAD,  PAD ]     ← 监督标签

tgt_input: (B, T)   dtype=torch.int64
tgt_y:     (B, T)   dtype=torch.int64
```

**物理意义**: 训练时用 Teacher Forcing。Decoder 看到 `<bos>` 时应该预测出 `89`，看到 `89` 时应该预测出 `203`，以此类推。

### 1.3 构造 tgt_mask

```python
pad_mask    = (tgt_input != pad_idx).unsqueeze(1)           # (B, 1, T)
causal_mask = torch.tril(torch.ones(1, T, T, dtype=bool))   # (1, T, T)
tgt_mask    = pad_mask & causal_mask                         # (B, T, T)  广播
```

```
causal_mask (下三角):          pad_mask:              tgt_mask (按位与):
┌─────────────────┐            ┌───────────┐          ┌─────────────────┐
│ True  False False│            │True       │          │ True  False False│
│ True  True  False│            │True       │          │ True  True  False│
│ True  True  True │            │True       │          │ True  True  True │
└─────────────────┘            └───────────┘          └─────────────────┘
       (1, T, T)                  (B, 1, T)                  (B, T, T)

当 T=6 且最后两个是 PAD 时:
tgt_mask[0] =
 [[ T, F, F, F, F, F],    ← 位置 0 (<bos>) 只能看自己
  [ T, T, F, F, F, F],    ← 位置 1 能看 0,1
  [ T, T, T, F, F, F],
  [ T, T, T, T, F, F],
  [ T, T, T, T, T, F],    ← 位置 4 (<eos>) 能看 0-4
  [ F, F, F, F, F, F]]    ← 位置 5 (PAD) 全部屏蔽
```

**作用**: 在 Decoder 自注意力中，同时实现因果遮蔽（不看未来）+ PAD 遮蔽。

### 1.4 统计有效 token 数

```python
ntokens = int((tgt_y != pad_idx).sum())   # 标量 int
```

**作用**: 用于 loss 归一化（除以有效 token 数而非 batch 大小）。

## 阶段 2：Embedding 层（nets/utils/encoder_decoder.py → Embeddings）

`src` 和 `tgt_input` 分别经过各自的 Embedding 层（若 `share_embeddings=True`，两者共享权重）。

### 2.1 Token Embedding

```python
x = self.lut(token_ids)   # (B, seq_len) → (B, seq_len, d_model)
```

```
src:      (B, S)  ──Embedding──→  (B, S, 512)
tgt_input:(B, T)  ──Embedding──→  (B, T, 512)
```

### 2.2 乘以 sqrt(d_model)

```python
x = self.lut(x) * math.sqrt(self.d_model)   # 乘以 √512 ≈ 22.627
```

**物理意义**: 论文原文的做法。Embedding 的初始化方差较小，乘以 sqrt(d_model) 使数值范围与位置编码匹配，避免位置信号淹没语义信号。

### 2.3 叠加位置编码（PositionalEncoding）

```python
# 预计算好的 buffer，不参与训练
pe: (1, max_len=5000, d_model)

x = x + self.pe[:, :seq_len]   # (B, seq_len, 512) + (1, seq_len, 512) → 广播
x = self.dropout(x)            # (B, seq_len, 512)
```

```
Positional Encoding (pe) 的构造:

position = [0, 1, 2, ..., max_len-1]       shape: (max_len, 1)
div_term = exp(arange(0, 512, 2) * -ln(10000)/512)   shape: (256,)

pe[:, 0::2] = sin(position * div_term)     偶数维度: sin
pe[:, 1::2] = cos(position * div_term)     奇数维度: cos

最终 pe: (1, 5000, 512)   注册为 buffer（不参与梯度更新）
```

**Embedding 层完整输出**:

```
src_embed_out:  (B, S, 512)   dtype=float32
tgt_embed_out:  (B, T, 512)   dtype=float32
```

## 阶段 3：Encoder（nets/utils/encoder_decoder.py → Encoder）

输入: `x = src_embed_out`，形状 `(B, S, 512)`

### 3.1 单层 EncoderLayer 内部（共 N=6 层）

每层的计算流程（Pre-LN 架构）:

```
输入 x: (B, S, 512)
    │
    ├── SublayerConnection[0] ── Self-Attention ──────────────────────────┐
    │   │                                                                 │
    │   ├── LayerNorm(x)         → (B, S, 512)                           │
    │   ├── MultiHeadedAttention(Q=norm_x, K=norm_x, V=norm_x, mask)    │
    │   │       │                                                         │
    │   │       ├── W_q(norm_x)  → (B, S, 512)                          │
    │   │       ├── W_k(norm_x)  → (B, S, 512)                          │
    │   │       ├── W_v(norm_x)  → (B, S, 512)                          │
    │   │       │                                                         │
    │   │       ├── reshape → (B, S, 8, 64)                              │
    │   │       ├── transpose → (B, 8, S, 64)                            │
    │   │       │                                                         │
    │   │       ├── scores = Q @ K^T / √64                               │
    │   │       │       → (B, 8, S, 64) @ (B, 8, 64, S)                 │
    │   │       │       → (B, 8, S, S)                                   │
    │   │       │                                                         │
    │   │       ├── mask → masked_fill(scores, mask==0, -1e9)            │
    │   │       │       → (B, 8, S, S)                                   │
    │   │       │                                                         │
    │   │       ├── softmax(scores, dim=-1) → attn_weights               │
    │   │       │       → (B, 8, S, S)                                   │
    │   │       │                                                         │
    │   │       ├── dropout(attn_weights)                                │
    │   │       │                                                         │
    │   │       ├── attn_weights @ V                                     │
    │   │       │       → (B, 8, S, S) @ (B, 8, S, 64)                  │
    │   │       │       → (B, 8, S, 64)                                  │
    │   │       │                                                         │
    │   │       ├── transpose → (B, S, 8, 64)                            │
    │   │       ├── reshape → (B, S, 512)                                │
    │   │       └── W_o(concat) → (B, S, 512)                            │
    │   │                                                                 │
    │   └── x = x + dropout(attn_out)   → (B, S, 512)   [残差连接]       │
    │                                                                     │
    ├── SublayerConnection[1] ── Feed-Forward ──────────────────────────-─┘
    │   │
    │   ├── LayerNorm(x)         → (B, S, 512)
    │   ├── FFN:
    │   │       ├── W_1(norm_x)  → (B, S, 2048)     [升维]
    │   │       ├── ReLU()       → (B, S, 2048)
    │   │       ├── dropout()    → (B, S, 2048)
    │   │       └── W_2(x)       → (B, S, 512)      [降维]
    │   │
    │   └── x = x + dropout(ffn_out)     → (B, S, 512)   [残差连接]
    │
    └── 输出 x: (B, S, 512)  → 送入下一层
```

### 3.2 Encoder 堆叠 6 层后

```python
for layer in self.layers:     # 6 个 EncoderLayer
    x = layer(x, mask)
return self.norm(x)           # 最终 LayerNorm
```

```
Encoder 最终输出 (memory): (B, S, 512)   dtype=float32
```

## 阶段 4：Decoder（nets/utils/encoder_decoder.py → Decoder）

输入:
- `x = tgt_embed_out`，形状 `(B, T, 512)`
- `memory = Encoder 输出`，形状 `(B, S, 512)`

### 4.1 单层 DecoderLayer 内部（共 N=6 层）

```
输入 x: (B, T, 512),  memory: (B, S, 512)
    │
    ├── SublayerConnection[0] ── Masked Self-Attention ────────────────────┐
    │   │                                                                  │
    │   ├── LayerNorm(x)                                                  │
    │   ├── MultiHeadedAttention(Q=x, K=x, V=x, mask=tgt_mask)           │
    │   │       │                                                          │
    │   │       ├── Q, K, V 各自 Linear(512,512) → reshape → (B,8,T,64)  │
    │   │       │                                                          │
    │   │       ├── scores = Q @ K^T / √64                                │
    │   │       │       → (B, 8, T, T)                                    │
    │   │       │                                                          │
    │   │       ├── masked_fill(tgt_mask==0, -1e9)    ← 因果+PAD 双重遮蔽  │
    │   │       │       → (B, 8, T, T)                                    │
    │   │       │                                                          │
    │   │       ├── softmax → dropout → @ V → (B, 8, T, 64)              │
    │   │       └── reshape → Linear → (B, T, 512)                        │
    │   │                                                                  │
    │   └── x = x + dropout(attn_out)   → (B, T, 512)                    │
    │                                                                      │
    ├── SublayerConnection[1] ── Cross-Attention ──────────────────────────┤
    │   │                                                                  │
    │   ├── LayerNorm(x)                                                  │
    │   ├── MultiHeadedAttention(Q=x, K=memory, V=memory, mask=src_mask)  │
    │   │       │                                                          │
    │   │       ├── Q = W_q(x)      → (B, 8, T, 64)   ← 来自 Decoder     │
    │   │       ├── K = W_k(memory)  → (B, 8, S, 64)   ← 来自 Encoder     │
    │   │       ├── V = W_v(memory)  → (B, 8, S, 64)   ← 来自 Encoder     │
    │   │       │                                                          │
    │   │       ├── scores = Q @ K^T / √64                                │
    │   │       │       → (B, 8, T, S)    ← 注意: T×S，不是 T×T           │
    │   │       │                                                          │
    │   │       ├── masked_fill(src_mask==0, -1e9)    ← 仅 PAD 遮蔽       │
    │   │       │       → (B, 8, T, S)                                    │
    │   │       │                                                          │
    │   │       ├── softmax → dropout → @ V → (B, 8, T, 64)              │
    │   │       └── reshape → Linear → (B, T, 512)                        │
    │   │                                                                  │
    │   └── x = x + dropout(attn_out)   → (B, T, 512)                    │
    │                                                                      │
    ├── SublayerConnection[2] ── Feed-Forward ─────────────────────────────┤
    │   │                                                                  │
    │   ├── LayerNorm(x)                                                  │
    │   ├── FFN: W_1 → ReLU → dropout → W_2                              │
    │   │       (B, T, 512) → (B, T, 2048) → (B, T, 512)                 │
    │   │                                                                  │
    │   └── x = x + dropout(ffn_out)   → (B, T, 512)                     │
    │                                                                      │
    └── 输出 x: (B, T, 512) → 送入下一层
```

### 4.2 Decoder 堆叠 6 层后

```python
for layer in self.layers:     # 6 个 DecoderLayer
    x = layer(x, memory, src_mask, tgt_mask)
return self.norm(x)           # 最终 LayerNorm
```

```
Decoder 最终输出 (hidden_states): (B, T, 512)   dtype=float32
```

## 阶段 5：Generator 投影头

```python
logits = model.generator(hidden_states)   # Linear(512, vocab_size, bias=False)
```

```
hidden_states: (B, T, 512)  ──Linear(512, 32000)──→  logits: (B, T, 32000)
```

**注意**: 这里的 `generator.proj.weight` 与 `tgt_embed.lut.weight` 共享同一块内存（权重绑定）。

## 阶段 6：损失计算（LabelSmoothingLoss）

```python
loss = criterion(logits, tgt_y)
```

```
logits: (B, T, 32000)    ← 模型输出的原始分数
tgt_y:  (B, T)           ← 监督标签 token id

内部流程:
1. log_probs = log_softmax(logits, dim=-1)     → (B, T, 32000)
2. nll_loss = -log_probs.gather(tgt_y)          → (B, T)        ← 交叉熵
3. smooth_loss = -log_probs.mean(dim=-1)        → (B, T)        ← 均匀分布惩罚
4. loss = (1-smoothing) * nll_loss + smoothing * smooth_loss
5. loss = loss[tgt_y != pad_idx].mean()          ← 只对非 PAD token 求平均
```

## 全流程 Shape 总览

```
┌─────────────────────────────────────────────────────────────────┐
│  BPE 文本                                                        │
│  "A dog is running"  →  [156, 4821, 87, 3012, 2]               │
└────────────────────────────┬────────────────────────────────────┘
                             ↓
┌────────────────────────────┴────────────────────────────────────┐
│  DataLoader: padding + batching                                  │
│  src:       (B, S)         int64                                 │
│  tgt:       (B, T_full)    int64                                 │
└────────────────────────────┬────────────────────────────────────┘
                             ↓
┌────────────────────────────┴────────────────────────────────────┐
│  Seq2SeqBatch.from_tensors()                                     │
│  ├─ src_mask:    (B, 1, S)       bool                            │
│  ├─ tgt_input:   (B, T)          int64   (tgt[:, :-1])          │
│  ├─ tgt_y:       (B, T)          int64   (tgt[:, 1:])           │
│  ├─ tgt_mask:    (B, T, T)       bool   (causal & pad)          │
│  └─ ntokens:     scalar          int                             │
└────────────────────────────┬────────────────────────────────────┘
                             ↓
┌────────────────────────────┴────────────────────────────────────┐
│  Embedding (src)                    Embedding (tgt)              │
│  (B, S) → ×√512 → +PE → dropout  (B, T) → ×√512 → +PE → drop │
│  输出: (B, S, 512)               输出: (B, T, 512)              │
└────────┬───────────────────────────────┬────────────────────────┘
         ↓                               ↓
┌────────┴──────────────┐    ┌───────────┴────────────────────────┐
│  Encoder × 6 层       │    │  Decoder × 6 层                    │
│  每层:                │    │  每层:                              │
│  ├─ LN               │    │  ├─ LN                             │
│  ├─ Self-Attn        │    │  ├─ Masked Self-Attn               │
│  │  Q,K,V: (B,S,512) │    │  │  Q,K,V: (B,T,512)              │
│  │  scores: (B,8,S,S)│    │  │  scores: (B,8,T,T)             │
│  │  output: (B,S,512)│    │  │  output: (B,T,512)             │
│  ├─ Residual+Dropout │    │  ├─ Residual+Dropout               │
│  ├─ LN               │    │  ├─ LN                             │
│  ├─ FFN              │    │  ├─ Cross-Attn (Q from dec, KV from enc)│
│  │  (B,S,512→2048→512)│   │  │  Q: (B,T,512) K,V: (B,S,512)  │
│  └─ Residual+Dropout │    │  │  scores: (B,8,T,S)             │
│                      │    │  │  output: (B,T,512)             │
│  memory: (B, S, 512) │    │  ├─ Residual+Dropout               │
│                      │    │  ├─ LN                             │
│                      │    │  ├─ FFN                            │
│                      │    │  │  (B,T,512→2048→512)             │
│                      │    │  └─ Residual+Dropout               │
│                      │    │                                    │
│                      │    │  hidden: (B, T, 512)               │
└──────────┬───────────┘    └──────────────┬─────────────────────┘
           │                               │
           │   memory (B, S, 512) ─────────┘
           │                               ↓
           │               ┌───────────────┴──────────────────────┐
           │               │  Generator                            │
           │               │  Linear(512, 32000, bias=False)       │
           │               │  (B, T, 512) → (B, T, 32000)         │
           │               └───────────────┬──────────────────────┘
           │                               ↓
           │               ┌───────────────┴──────────────────────┐
           │               │  LabelSmoothingLoss                   │
           │               │  logits: (B, T, 32000)                │
           │               │  tgt_y:  (B, T)                       │
           │               │  → loss: scalar                       │
           │               └──────────────────────────────────────┘
```

## 关键 Shape 变化速查表

| 位置 | 输入 Shape | 输出 Shape | 操作 |
|------|-----------|-----------|------|
| Token Embedding | (B, seq_len) | (B, seq_len, 512) | nn.Embedding 查表 |
| × sqrt(d_model) | (B, seq_len, 512) | (B, seq_len, 512) | 逐元素乘法 |
| + Positional Encoding | (B, seq_len, 512) | (B, seq_len, 512) | 逐元素加法 |
| W_q / W_k / W_v | (B, seq_len, 512) | (B, seq_len, 512) | Linear(512, 512) |
| 多头切分 | (B, seq_len, 512) | (B, 8, seq_len, 64) | reshape + transpose |
| Q @ K^T | (B,8,S,64)@(B,8,64,S) | (B, 8, S, S) | 批量矩阵乘法 |
| softmax | (B, 8, S, S) | (B, 8, S, S) | 概率归一化 |
| attn @ V | (B,8,S,S)@(B,8,S,64) | (B, 8, S, 64) | 批量矩阵乘法 |
| 多头合并 | (B, 8, S, 64) | (B, S, 512) | transpose + reshape |
| W_o | (B, S, 512) | (B, S, 512) | Linear(512, 512) |
| FFN W_1 | (B, S, 512) | (B, S, 2048) | Linear(512, 2048) |
| FFN W_2 | (B, S, 2048) | (B, S, 512) | Linear(2048, 512) |
| Generator | (B, T, 512) | (B, T, 32000) | Linear(512, 32000) |
| Cross-Attn scores | (B,8,T,64)@(B,8,64,S) | (B, 8, T, S) | Decoder Q × Encoder KV |
